Tables

Sections in a table:

  1. Table head <<thead>>
    • Table Row   <tr>
    • Table Heading <th>
  2. Table body <body>
    • Table Row   <tr>
    • Table Heading <th>
    • Table Data     <td>

Table Head <<thead>>:

Such a section of table which contains all the variable attributes for which various possible outcomes are possible,e.g; for Name(Ahmad, Ali, etc) is called Table Head

Table Body <tbody>:

Such a section of table which contains different values for a single attribute,e.g; for Name(Ahmad, Ali, etc) is called Table Body.

  1. Table Row   <tr>:

Example Snippet:

<table class="table">
 <thead>
  <tr>
    <th scope="col">#</th>
    <th scope="col">First</th>
    <th scope="col">Last</th>
    <th scope="col">Handle</th>
    </tr>
 </thead>
 <tbody>
  <tr>
    <th scope="row">1</th>
    <td>Mark</td>
    <td>Otto</td>
    <td>@mdo</td>
  </tr>
  <tr>
    <th scope="row">2</th>
    <td>Jacob</td>
    <td>Thornton</td>
    <td>@fat</td>
  </tr>
  <tr>
    <th scope="row">3</th>
    <td colspan="2">Larry the Bird</td>
    <td>@twitter</td>
  </tr>
 </tbody>
</table>
