Striped tables in Accented tables

These are those tables in which alternative rows are given opacity, the syntax for these kind of tables is class="table table-striped"

Striped columns in accented tables

By default table-striped class acts on rows not on columns, so in order to have striped columns, we'll use class="table table-striped-columns"

Hoverable Tables

Hoverable Tables are mostly used to provide a visual respose to the user, it can also be used for editable tables and also in large datasets for data comparison. The syntax for this is class="table table-hover"

Active tables

This utility class is mainly used:

  1. To highlight that specific row or cell of the table which is being recently clicked.
  2. To indicate cells of the row from which the user is navigating using keyboard.
  3. To indicate specific cells in dataset that meet a specific condition.
  4. To represent differnt entities in row.

Its's syntax is: class="table table-active"
